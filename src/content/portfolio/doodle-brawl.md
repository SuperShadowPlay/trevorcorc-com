[Doodle Brawl](https://doodlebrawl.com/) was built with a front-end in React JS and a back-end in Python, Doodle Brawl is an AI-based game where users submit drawings to be evaluated by Google's Gemini API for their fighting prowess. Gemini then simulates a play-by-play of a fight between two characters and shows spectators the results. Doodlebrawl started as a submission to my school's 2026 24-hour hackathon event, but development has continued far past that first day!
