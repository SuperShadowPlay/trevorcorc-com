Build, fight, build, repeat! [Fusion Fighters](https://dougisdoug.itch.io/fusion-fighters) is a 2D party-style fighter has you progressively customizing your character over each new round. Try all the combinations! Fusion Fighters was a project that was created in Unity for my school's Game Development class.
